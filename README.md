**RetroGames-for-Android** adalah aplikasi emulator ringan yang dirancang untuk membawa kenangan masa kecil langsung ke genggaman Anda. Nikmati ribuan game klasik dari berbagai konsol retro legendaris dengan performa yang dioptimalkan untuk perangkat Android modern.

Aplikasi ini fokus pada kesederhanaan, kecepatan, dan kompatibilitas tinggi, memungkinkan Anda menjalankan judul-judul favorit tanpa hambatan teknis yang rumit.

* **Kompatibilitas Luas:** Mendukung berbagai format ROM dari konsol populer (NES, SNES, GBA, dan lainnya).

* **Hemat Baterai:** Efisiensi pemrosesan yang memastikan Anda bisa bermain lebih lama tanpa cepat menguras daya.

## ğŸ“¥ Instalasi

Ikuti langkah-langkah sederhana ini untuk mulai bermain:

1.  **Unduh File APK:** Klik tautan di bawah ini untuk mengunduh versi terbaru:
    > [**Download RetroGames v1.0 (APK)**](https://github.com/PurwanaNet/retrogames-android/releases/download/1.0/retrogames.apk)
2.  **Izinkan Sumber Tidak Dikenal:** Buka *Settings* > *Security* pada perangkat Anda, dan aktifkan opsi "Unknown Sources" atau "Install Unknown Apps".
3.  **Instal APK:** Buka file yang telah diunduh melalui File Manager dan pilih "Install".
4.  **Siapkan ROM:** Pindahkan file game (ROM) Anda ke penyimpanan internal ponsel.
5.  **Mainkan:** Buka aplikasi, scan folder ROM, dan selamat bernostalgia!
